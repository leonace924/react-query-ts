Technical React Questions
- What are props in React?
  Props are the objects which are used for passing data from one component to another, like from parent to children.

- What is state in React?
  State is the object that stores and manages component's own data.

- What is the difference between state and props?
  State is mutable and controlled by the component itself internally, and props are immutable and are set by parent component.
